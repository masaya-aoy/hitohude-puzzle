一筆書きロジックパズル

Web Application Portfolio

アプリ概要

「すべてのマスを一筆書きで通過する」というハミルトン閉路問題に、「指定された順序で特定のマスを通過する」という制約を組み合わせたオリジナルのロジックパズルゲームです。

要件定義からUX設計（スマホでの指の離れへの対応など）、ゲームバランスの調整までを一貫して行い、ブラウザ上で軽快に動作するようJavaScriptで実装しました。

実装した主な機能・工夫点：
  • タッチデバイスに最適化された直感的なドラッグ操作
  • 進行状況の自動セーブ機能（Local Storage活用）
  • 間違えた箇所を即座に視覚化するリアルタイム判定
  • 「◯番目以外」「◯番目以下」などの複雑な不等号制約ロジック（応用編）
